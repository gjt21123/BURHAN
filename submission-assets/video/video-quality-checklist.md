# Video quality checklist

- [ ] Duration is below 3:00.
- [ ] Voiceover matches `final-voiceover.txt` and final sentence is exact.
- [ ] `LIVE CODEX RUN`, `LIVE BURHAN VERIFICATION`, and `DETERMINISTIC REPAIR DEMO` are readable.
- [ ] Live historical evidence and deterministic repair are never conflated.
- [ ] No live GPT-5.6 inference is claimed.
- [ ] No credentials, paths, terminals, prompts, raw JSONL/stderr, hidden validator source, or private reasoning is visible.
- [ ] Captions match the actual spoken narration.
- [ ] Reset returns to the original rejected state at the end.
- [ ] Footage and narration files exist before any MP4 assembly attempt.
